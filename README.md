# Data Engineering Zoomcamp - Homework

This repository contains homework assignments and projects for the Data Engineering Zoomcamp course.

## About

This repository is organized to track progress through the Data Engineering Zoomcamp, covering various aspects of modern data engineering including:

- Data ingestion
- Workflow orchestration
- Data warehousing
- Analytics engineering
- Batch processing
- Streaming

## Repository Structure

```
.
├── week_1_basics_n_setup/
├── week_2_workflow_orchestration/
├── week_3_data_warehouse/
├── week_4_analytics_engineering/
├── week_5_batch_processing/
├── week_6_streaming/
└── project/
```

## Prerequisites

- Docker and Docker Compose
- Python 3.9+
- PostgreSQL
- Google Cloud Platform account
- Terraform

## Setup

1. Clone this repository:
```bash
git clone https://github.com/josephArinaitwe/De-zoomcamp-homework.git
cd De-zoomcamp-homework
```

2. Set up your environment variables as needed for each module

3. Follow the instructions in each week's folder for specific setup requirements
